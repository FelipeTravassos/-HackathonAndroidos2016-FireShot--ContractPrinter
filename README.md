[HackathonAndroidos2016-FireShot] Contract Printer
================

Aplicativo desenvolvido para samsung smart-printing. Contract Printer é uma ferramenta que agiliza a impressão de contratos e arquivos comuns da empresa. O usuário poderá usar os templates que já estão cadastrados ou importar novos templates.
